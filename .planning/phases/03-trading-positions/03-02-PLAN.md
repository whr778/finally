   ```

   ```
   <div className="flex-1 flex items-center justify-center">
     <p className="text-xs text-text-muted">No positions yet. Execute a trade to get started.</p>
   </div>
   ```

   <tr key={p.ticker} data-testid={`position-row-${p.ticker}`}>
   ```

   Column cells (all `className="font-mono text-sm px-2 py-1"`):

   - %: `text-right ${pnlColor(p.pct_change)}` — display formatted percent:
     - If `p.pct_change === null`: display `'—'`
     - If positive: `'+' + p.pct_change.toFixed(2) + '%'`
     - If negative: `p.pct_change.toFixed(2) + '%'` (toFixed includes the minus sign)
     - If zero: `'0.00%'`
