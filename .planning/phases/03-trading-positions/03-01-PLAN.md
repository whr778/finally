Rewrite `frontend/src/components/trade-bar.tsx` to activate trade execution. The complete implementation:

1. **Imports:** Add `useState` and `useEffect` from react. Import `usePortfolioStore` from `@/stores/portfolio-store`.

2. **Local state:** Add three state variables:
   - `const [ticker, setTicker] = useState(selectedTicker || '')`
   - `const [qty, setQty] = useState('')`
   - `const [feedback, setFeedback] = useState<{ type: 'success' | 'error'; message: string } | null>(null)`

3. **Prop sync (per D-06):** Add useEffect to sync selectedTicker prop to local ticker state:
   ```
   useEffect(() => {
     if (selectedTicker) setTicker(selectedTicker)
   }, [selectedTicker])
   ```

4. **Store selector:** `const fetchPortfolio = usePortfolioStore((s) => s.fetchPortfolio)`

5. **handleTrade function:** Create `async function handleTrade(side: 'buy' | 'sell')`:
   - Validate ticker: `if (!ticker.trim())` -> set feedback `{ type: 'error', message: 'Enter a ticker symbol' }` and return
   - Validate qty: `const quantity = parseFloat(qty)` then `if (!qty || isNaN(quantity) || quantity <= 0)` -> set feedback `{ type: 'error', message: 'Enter a quantity' }` and return
   - Call `fetch('/api/portfolio/trade', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ ticker: ticker.trim().toUpperCase(), quantity, side }) })`
   - If `!res.ok`: `const err = await res.json()` then set feedback `{ type: 'error', message: err.detail || 'Trade failed' }` and return
   - On success: `const trade = await res.json()` then:
     - Set feedback: `{ type: 'success', message: \`${trade.side.toUpperCase()} ${trade.quantity} ${trade.ticker} @ $${trade.price.toFixed(2)}\` }` (per D-01, D-02)
     - Clear qty: `setQty('')` (per D-05: keep ticker, clear qty on success)
     - Refresh portfolio: `fetchPortfolio()` (per D-08: immediate refresh)
   - Wrap in try/catch: catch block sets feedback `{ type: 'error', message: 'Trade failed. Try again.' }`

6. **Auto-dismiss effect:** Add useEffect that clears feedback after 3 seconds:
   ```
   useEffect(() => {
     if (!feedback) return
     const timer = setTimeout(() => setFeedback(null), 3000)
     return () => clearTimeout(timer)
   }, [feedback])
   ```

7. **JSX structure:** Change the outer div from `flex items-center` to `flex flex-col`:

   - Second child: the feedback row (conditionally rendered)
