**P&L color scale function -- returns `{ color, opacity }` based on pct_change (per D-01, UI-SPEC color table):**
```typescript
function tileColor(pctChange: number): { color: string; opacity: number } {
  const abs = Math.abs(pctChange)
  if (abs < 0.5) return { color: '#7d8590', opacity: 0.3 }
  if (pctChange > 0) {
    if (pctChange >= 5) return { color: '#3fb950', opacity: 0.8 }
    if (pctChange >= 2) return { color: '#3fb950', opacity: 0.6 }
    return { color: '#3fb950', opacity: 0.4 }
  }
  if (pctChange <= -5) return { color: '#f85149', opacity: 0.8 }
  if (pctChange <= -2) return { color: '#f85149', opacity: 0.6 }
  return { color: '#f85149', opacity: 0.4 }
}
```

**P&L percentage formatter (same format as positions-table.tsx):**
```typescript
function formatPct(value: number): string {
  if (value === 0) return '0.00%'
  if (value > 0) return '+' + value.toFixed(2) + '%'
  return value.toFixed(2) + '%'
}
```

**Custom tile content renderer -- passed to Treemap `content` prop.** This is a function component that Recharts calls with props `{ x, y, width, height, name, pct_change }`:
```typescript
function CustomTile(props: any) {
  const { x, y, width, height, name, pct_change } = props
  if (!name || width <= 0 || height <= 0) return null
  const { color, opacity } = tileColor(pct_change ?? 0)
  const showPct = width >= 40 && height >= 30

  return (
    <g data-testid={`tile-${name}`} onClick={() => onSelectTicker(name)} style={{ cursor: 'pointer' }}>
      <rect
        x={x}
        y={y}
        width={width}
        height={height}
        fill={color}
        fillOpacity={opacity}
        stroke="rgba(0,0,0,0.3)"
        strokeWidth={1}
      />
      <text
        x={x + 8}
        y={y + 18}
        fill="#e6edf3"
        fontSize={14}
        fontFamily="'JetBrains Mono', monospace"
      >
        {name}
      </text>
      {showPct && (
        <text
          x={x + 8}
          y={y + 34}
          fill="#e6edf3"
          fontSize={12}
          fontFamily="'Inter', system-ui, sans-serif"
        >
          {formatPct(pct_change ?? 0)}
        </text>
      )}
    </g>
  )
}
```

IMPORTANT: The `onSelectTicker` reference inside `CustomTile` means `CustomTile` must be defined inside the component function body (closure over props) or accept `onSelectTicker` through the Recharts data items. The cleanest approach: define `CustomTile` inside the `HeatmapPanel` component so it closes over `onSelectTicker`.

**Recharts Treemap JSX:**

NOTE: Recharts Treemap `content` prop receives a React element. The library clones it and passes tile props. Define `CustomTile` as a component and pass `<CustomTile />` as the content prop value.

**Fetch + polling (per D-09, D-17) -- fetch on mount, poll every 30 seconds:**
```typescript
const [snapshots, setSnapshots] = useState<Snapshot[]>([])

useEffect(() => {
  let active = true
  async function load() {
    const res = await fetch('/api/portfolio/history')
    if (res.ok && active) {
      const data: Snapshot[] = await res.json()
      setSnapshots(data)
    }
  }
  load()
  const interval = setInterval(load, 30_000)
  return () => { active = false; clearInterval(interval) }
}, [])
```

function formatTime(value: string): string {

}

function formatDollar(value: number): string {
