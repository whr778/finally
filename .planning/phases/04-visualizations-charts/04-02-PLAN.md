**Imports:**

interface TickerChartProps {

**Component body:**

The component manages three refs:
1. `chartContainerRef` -- a `useRef<HTMLDivElement>(null)` for the DOM element where Lightweight Charts mounts
2. `chartRef` -- a `useRef<IChartApi | null>(null)` for the chart instance
3. `seriesRef` -- a `useRef<ISeriesApi<'Area'> | null>(null)` for the area series instance
4. `dataRef` -- a `useRef<{ time: UTCTimestamp; value: number }[]>([])` for accumulated price points

**Chart creation/destruction -- useEffect depending on selectedTicker:**

When `selectedTicker` changes:
1. If a chart already exists, call `chartRef.current.remove()` to destroy it, reset refs
2. If `selectedTicker` is null, do nothing (show empty state)
3. If `selectedTicker` is truthy, create the chart:

```typescript
const chart = createChart(chartContainerRef.current!, {
  width: chartContainerRef.current!.clientWidth,
  height: chartContainerRef.current!.clientHeight,
  layout: {
    background: { type: 'solid', color: '#161b22' },
    textColor: '#7d8590',
    fontFamily: "'Inter', system-ui, sans-serif",
    fontSize: 12,
  },
  grid: {
    vertLines: { color: 'rgba(125, 133, 144, 0.1)' },
    horzLines: { color: 'rgba(125, 133, 144, 0.1)' },
  },
  timeScale: {
    timeVisible: true,
    secondsVisible: false,
    borderColor: 'rgba(125, 133, 144, 0.2)',
  },
  rightPriceScale: {
    borderColor: 'rgba(125, 133, 144, 0.2)',
  },
  crosshair: {
    mode: 0,
    vertLine: { width: 1, color: 'rgba(125, 133, 144, 0.4)', style: 3 },
    horzLine: { width: 1, color: 'rgba(125, 133, 144, 0.4)', style: 3 },
  },
})

const areaSeries = chart.addSeries(AreaSeries, {
  lineColor: '#209dd7',
  lineWidth: 1.5,
  topColor: 'rgba(32, 157, 215, 0.3)',
  bottomColor: 'rgba(32, 157, 215, 0)',
  lastValueVisible: true,
  priceLineVisible: false,
  crosshairMarkerVisible: true,
  crosshairMarkerRadius: 3,
})

chartRef.current = chart
seriesRef.current = areaSeries
dataRef.current = []
```

4. Set up a ResizeObserver to call `chart.resize(width, height)` on container resize
5. Return a cleanup function that removes the ResizeObserver and calls `chart.remove()`

**Price data accumulation -- useEffect depending on priceData from SSE:**

Subscribe to the selected ticker's price data using `useTickerPrice`:
```typescript
const priceData = useTickerPrice(selectedTicker ?? '')
```

When `priceData` updates AND `seriesRef.current` exists AND `selectedTicker` is truthy:
1. Create a new point: `{ time: priceData.timestamp as UTCTimestamp, value: priceData.price }`
2. Push to `dataRef.current`
3. Call `seriesRef.current.update(point)` to append the point to the chart (NOT `setData` -- `update` is more efficient for streaming)
4. On the first few data points (e.g., `dataRef.current.length <= 3`), call `chartRef.current.timeScale().fitContent()` to auto-fit the visible range

IMPORTANT: Lightweight Charts requires that time values are strictly increasing. The SSE `timestamp` field is in seconds since epoch. If two updates arrive with the same second, only update the last point (Lightweight Charts handles this via `update` -- if the time matches an existing point, it updates in place rather than creating a duplicate).
